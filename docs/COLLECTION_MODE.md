# 動詞図鑑 (Verb Collection) 仕様 (Specification)

## 概要 (Overview)

学習のモチベーション維持と、動詞ごとの語法定着を目的としたゲーミフィケーション機能です。
ユーザーは学習を通じて動詞カードを獲得・育成し、コレクションを完成させます。

## カードシステム (Card System)

### 1. 獲得条件 (Unlock)
SniperモードまたはReviewモードで、特定の動詞タグ (`v:verbId`) が付いた問題に正解すると、その動詞カードがアンロックされます。

### 2. レアリティ (Rarity)
動詞の重要度やパターンの複雑さに応じてレアリティを設定しています。

| レアリティ | 条件例 | 色 |
| :--- | :--- | :--- |
| **N (Normal)** | 基本的な動詞 (パターンが1-2個) | グレー |
| **R (Rare)** | 重要動詞、またはパターンが3つ以上 | シルバー |
| **SR (Super Rare)** | 非常に多くのパターンを持つ、または混同しやすい重要動詞 | ゴールド |

### 3. レベルと経験値 (Level & XP)
カード獲得後、その動詞に関連する問題に正解するたびに **10 XP** を獲得します。
一定のXPに達するとレベルアップします。

| レベル | 必要累積XP | 解放要素 (将来構想) |
| :--- | :--- | :--- |
| Lv.1 | 0 | カードアンロック |
| Lv.2 | 50 | (例: 音声解放) |
| Lv.3 | 150 | (例: 詳細解説解放) |
| Max | 300 | マスター称号 |

## データ構造 (Data Structure)

動詞ごとの進捗は `VerbCardCollection` として `localStorage` に保存されます (Key: `e5pt.verbcards.v1`)。
詳細な型定義は `src/domain/types.ts` を参照してください。
