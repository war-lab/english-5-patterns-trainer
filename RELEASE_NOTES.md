# Release Notes

## v1.2.0 - Verb Collection Mode Update

### 🚀 新機能 (New Features)

*   **動詞図鑑 (Verb Collection Mode)**
    *   学習した動詞を「カード」として収集できるモードを追加しました。
    *   **カード獲得**: スナイパーモードや復習モードで正解すると、その動詞のカードを獲得できます。
    *   **育成要素**: カードを獲得した後も、その動詞の問題に正解することで経験値 (XP) が入り、レベルアップします。
    *   **個別トレーニング**: 図鑑の詳細画面から、その動詞が出題される問題だけを絞り込んでトレーニング ("Train") できます。

*   **スナイパーモードの機能拡張**
    *   `deck` パラメータによるフィルタリングに対応しました (例: `?deck=v:give`)。
    *   正解時にカード獲得やレベルアップのポップアップ通知を表示するようにしました。

### 🔧 改善・変更 (Improvements & Changes)

*   **データ構造の刷新**
    *   問題データのタグ形式を刷新し、動詞 (`v:verb`) と文型 (`p:pattern`) を明確に区別するようにしました。
    *   これにより、より正確な「苦手動詞」の分析が可能になります。
*   **トップページのUI改善**
    *   メニューに「動詞図鑑」を追加しました。

### 🐛 バグ修正 (Bug Fixes)

*   古いバージョンの回答データが存在する場合に、統計計算でエラーが発生しトップページが開かなくなる問題を修正しました。
*   Home画面の文型ラベル表示における軽微な不具合を修正しました。
